# Contexto de negocio

En distintas áreas de apoyo, parte del trabajo operativo dependía de tareas repetitivas realizadas manualmente sobre archivos Excel. Estas tareas consumían tiempo, aumentaban el riesgo de error y dificultaban la entrega rápida de información procesada.

Frente a este escenario, se desarrollaron scripts en Python orientados a automatizar procesos específicos de lectura, transformación y consolidación de datos, entregando resultados más rápidos, consistentes y reutilizables.
